# Skins for use on FJP Shadowlands DCS server

## How to download
Click the green code button above then click Download ZIP

**or**

Go to the [releases](https://github.com/SteveLockeUK/FJP_DCS_Liveries/releases/latest) page and download the zip for the latest release.

## How to use
The skins need to be placed in the Liveries directory in your DCS saved games folder.
1. Download and extract the latest .zip
1. Copy the Liveries folder extracted in the previous step into `%USERPROFILE%\Saved Games\DCS.openbeta`
